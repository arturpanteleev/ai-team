## ДОБАВЛЕННЫЕ Требования

### Requirement: AgentCLI Runtime
Система MUST реализовать `AgentCLIRuntime`, который делегирует выполнение AI-агентскому CLI.

#### Scenario: Выполнение через opencode
- **КОГДА** AgentCLIRuntime запускается
- **ТОГДА** система MUST собрать промпт из system prompt агента + содержимого входных артефактов
- **И** записать промпт во временный `.md` файл
- **И** запустить `opencode --resume --message-file <tempfile>` в директории целевого проекта
- **И** дождаться завершения

### Requirement: Проверка наличия OpenCode
Система MUST проверять, доступен ли настроенный CLI, перед выполнением.

#### Scenario: CLI не найден
- **КОГДА** настроенный CLI (например, `opencode`) не найден в PATH
- **ТОГДА** система MUST вернуть понятную ошибку: "opencode: команда не найдена. Установите с https://opencode.ai"

### Requirement: Настраиваемый CLI-бинарник
Система MUST поддерживать разные AI-агентские CLI через конфиг.

#### Scenario: Переключение на claude
- **КОГДА** в `.ai-team/config.yaml` указано `cli: claude`
- **ТОГДА** runtime MUST запускать `claude --resume --message-file <tempfile>` вместо `opencode`

### Requirement: Структура промпта
Файл промпта MUST включать: system prompt агента, контекст артефактов и явные инструкции по выводу.

#### Scenario: Формат промпта
- **КОГДА** создаётся файл промпта
- **ТОГДА** он MUST содержать:
  - Описание роли агента из prompt.md
  - Ссылки на входные артефакты (пути к файлам, не полное содержимое для больших файлов)
  - Чёткие ожидания по результату: какие файлы создать и куда
