## ДОБАВЛЕННЫЕ Требования

### Требование: Пакет pkg/eval
Система ДОЛЖНА предоставлять пакет для оценки качества артефактов.

#### Сценарий: Eval структура
- **КОГДА** создаётся `pkg/eval`
- **ТОГДА** он ДОЛЖЕН содержать тип `Eval` с полями AgentName, ArtifactPath, Criteria
- **И** тип `Result` с полями Score, Comment

#### Сценарий: Запуск оценки
- **КОГДА** вызывается `eval.Run()`
- **ТОГДА** он ДОЛЖЕН прочитать артефакт
- **И** запустить opencode с промптом-судьёй
- **И** распарсить оценку и комментарий

### Требование: Команда ai-team eval
CLI ДОЛЖЕН поддерживать команду `eval`.

#### Сценарий: Eval запуск
- **КОГДА** пользователь запускает `ai-team eval --feature "hello-api" --task "..." --agent analyst`
- **ТОГДА** система ДОЛЖЕНА запустить Analyst, затем оценить его output
- **И** вывести таблицу с результатами

#### Сценарий: Eval без пайплайна
- **КОГДА** пользователь запускает `ai-team eval --artifact reviews/hello-api/review.md`
- **ТОГДА** система ДОЛЖЕНА оценить только указанный артефакт без запуска пайплайна
