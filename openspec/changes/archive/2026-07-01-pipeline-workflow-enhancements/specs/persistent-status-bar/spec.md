## ADDED Requirements

### Requirement: Status bar внизу терминала
Система ДОЛЖНА показывать персистентную строку состояния внизу терминала.

#### Scenario: Status bar содержит проект, фичу, агента и прогресс
- **КОГДА** пайплайн запущен
- **ТОГДА** внизу терминала ДОЛЖНА отображаться строка: `[ai-team] <project> | <feature> | <agent> (N/M)`
- **И** строка ДОЛЖНА оставаться видимой после вывода логов
- **И** строка ДОЛЖНА обновляться после каждого завершённого этапа

### Requirement: Status bar не затирается логами
Система ДОЛЖНА использовать ANSI escape для сохранения/восстановления позиции курсора.

#### Scenario: Логи выше status bar
- **КОГДА** агент выводит логи (через `fmt.Print`, `fmt.Println`)
- **ТОГДА** статус-бар ДОЛЖЕН оставаться внизу терминала
- **И** логи ДОЛЖНЫ отображаться выше статус-бара

### Requirement: Отключение status bar
Status bar ДОЛЖЕН отключаться, если stdout не является терминалом.

#### Scenario: Нет status bar при перенаправлении
- **КОГДА** stdout перенаправлен в файл или pipe
- **ТОГДА** status bar НЕ ДОЛЖЕН отображаться
