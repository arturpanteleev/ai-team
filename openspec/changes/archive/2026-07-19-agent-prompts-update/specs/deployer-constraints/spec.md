## ADDED Requirements

### Requirement: Формат коммита
Deployer MUST соблюдать строгий формат коммит-сообщения.

#### Scenario: Формат commit message
- **КОГДА** deployer создаёт коммит
- **ТОГДА** commit message MUST начинаться с номера задачи (если есть)
- **И** содержать краткое описание изменения
- **И** быть не длиннее 10 слов после номера задачи
- **И** НЕ содержать упоминаний или атрибуции агента

#### Scenario: Без атрибуции
- **КОГДА** deployer формирует commit message
- **ТОГДА** сообщение MUST NOT содержать "Generated by", "AI-authored", "Co-authored-by" или подобные футеры

### Requirement: Формат PR
Deployer MUST соблюдать ограничения на описание PR.

#### Scenario: Длина PR
- **КОГДА** deployer создаёт PR
- **ТОГДА** описание PR MUST быть не длиннее 700 символов
- **И** MUST кратко объяснять: что изменено, зачем, как проверялось

### Requirement: Проверка условий перед деплоем
Deployer MUST проверять review.md и test-report.md перед выполнением.

#### Scenario: Условия не выполнены
- **КОГДА** review.md НЕ содержит APPROVED или test-report.md НЕ содержит PASS
- **ТОГДА** deployer MUST вернуть BLOCKED статус
- **И** описать: "Деплой заблокирован: {причина}"

### Requirement: Все артефакты на русском языке
Deployer MUST создавать PR description на русском языке.

#### Scenario: Язык PR
- **КОГДА** deployer создаёт PR
- **ТОГДА** описание PR MUST быть на русском языке
