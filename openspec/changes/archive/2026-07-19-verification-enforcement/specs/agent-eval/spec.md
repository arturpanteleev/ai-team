## MODIFIED Requirements

### Requirement: Пакет pkg/eval
Система MUST предоставлять пакет для оценки качества артефактов с захватом и парсингом вывода судьи.

#### Scenario: Запуск оценки
- **КОГДА** вызывается `eval.Run()`
- **ТОГДА** он MUST прочитать артефакт
- **И** запустить судью тем же каноническим вызовом CLI, что и пайплайн (`opencode run <prompt>`, без `--resume` — свежая сессия)
- **И** захватить stdout судьи (дублируя в консоль)
- **И** распарсить `**Оценка:** <1-10>` и `**Комментарий:** ...` в `Result{Score, Comment}`

#### Scenario: Оценка не распарсена
- **КОГДА** вывод судьи не содержит строки оценки
- **ТОГДА** `Run` MUST вернуть ошибку `не удалось распарсить оценку судьи`

#### Scenario: Eval с запуском одного агента
- **КОГДА** пользователь запускает `ai-team eval --feature X --task Y --agent analyst`
- **ТОГДА** система MUST запустить пайплайн, состоящий только из указанного агента
- **И** оценить его фактические выходные артефакты (пути из def.yaml)
