## MODIFIED Requirements

### Requirement: Проверка наличия git
Система MUST проверять, является ли target-проект git-репозиторием, и сообщать о невозможности guard-проверки.

#### Scenario: Проект без git
- **КОГДА** targetDir не является git-репозиторием
- **ТОГДА** система MUST вывести warning `git-репозиторий не найден — git diff guard пропущен`
- **И** guard MUST быть пропущен

#### Scenario: Ошибка git-команды (fail-closed)
- **КОГДА** git-команды guard-проверки завершаются ошибкой (не «есть изменения», а сбой)
- **ТОГДА** этап MUST завершиться ошибкой `git diff guard: {причина}`
- **И** пайплайн остановиться (guard MUST NOT молча пропускать при сбое)
