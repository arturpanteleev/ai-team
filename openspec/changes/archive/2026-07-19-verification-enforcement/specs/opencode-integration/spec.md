## MODIFIED Requirements

### Requirement: AgentCLI Runtime
Система MUST реализовать `AgentCLIRuntime`, который делегирует выполнение AI-агентскому CLI единым каноническим вызовом.

#### Scenario: Выполнение через opencode
- **КОГДА** AgentCLIRuntime запускается
- **ТОГДА** система MUST собрать промпт из system prompt агента + содержимого входных артефактов + служебной секции
- **И** запустить `opencode run [-m <model>] <prompt>` в директории целевого проекта
- **И** дождаться завершения (с учётом таймаута этапа)

#### Scenario: Передача модели
- **КОГДА** для агента задана модель (не пустая и не `auto`)
- **И** CLI — `opencode`
- **ТОГДА** система MUST передать флаг `-m <model>`
- **И** ЕСЛИ CLI неизвестен системе — модель НЕ передаётся, выводится предупреждение

#### Scenario: Логи агента
- **КОГДА** агент выполняется
- **ТОГДА** его stdout и stderr MUST дублироваться в файл `.ai-team/logs/{feature}/{agent}.log`
- **И** одновременно выводиться в консоль
