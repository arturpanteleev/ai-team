## MODIFIED Requirements

### Requirement: Уведомления в пайплайне
Пайплайн ДОЛЖЕН вызывать Notifier после каждого завершённого этапа и поддерживать stage_started events.

#### Scenario: Notify вызывается после каждого этапа
- **КОГДА** агент завершает выполнение
- **ТОГДА** пайплайн ДОЛЖЕН вызвать `Notifier.Notify()` для всех зарегистрированных notifier-ов
- **И** передать StageResult: имя агента, статус, сообщение, время

#### Scenario: Stage started event
- **КОГДА** пайплайн начинает выполнение агента
- **ТОГДА** пайплайн ДОЛЖЕН уведомить web backend (если подключён) о начале этапа
