## ADDED Requirements

### Requirement: Команда ai-team web
Система ДОЛЖНА предоставить CLI команду для запуска web-сервера.

#### Scenario: Запуск с портом
- **КОГДА** пользователь запускает `ai-team web --port 9090`
- **ТОГДА** сервер ДОЛЖЕН запуститься на порту 9090

#### Scenario: Запуск по умолчанию
- **КОГДА** пользователь запускает `ai-team web`
- **ТОГДА** сервер ДОЛЖЕН запуститься на порту 8080

#### Scenario: Информация при запуске
- **КОГДА** сервер запускается
- **ТОГДА** в консоль ДОЛЖНО вывестись: `Web UI available at http://localhost:8080`
