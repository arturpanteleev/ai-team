## ADDED Requirements

### Requirement: ANSI-цветной вывод
Консольный вывод системы ДОЛЖЕН использовать ANSI-цвета для ключевых элементов.

#### Scenario: Цветной вывод статусов
- **КОГДА** система выводит статус этапа в терминал
- **ТОГДА** `✓` ДОЛЖЕН быть зелёным (`\033[32m`)
- **И** `✗` ДОЛЖЕН быть красным (`\033[31m`)
- **И** название этапа ДОЛЖНО быть жёлтым (`\033[33m`)

### Requirement: Автоотключение цветов
Цвета ДОЛЖНЫ автоматически отключаться при перенаправлении вывода.

#### Scenario: Цвета не выводятся в файл
- **КОГДА** stdout перенаправлен в файл или pipe
- **ТОГДА** ANSI-escape последовательности НЕ ДОЛЖНЫ выводиться
- **И** весь текст ДОЛЖЕН выводиться как plain text
