## ДОБАВЛЕННЫЕ Требования

### Требование: Coder реализует код по техническому дизайну
Агент Coder ДОЛЖЕН прочитать технический дизайн и реализовать код в целевом проекте.

#### Сценарий: Coder запускается
- **КОГДА** Coder запускается
- **ТОГДА** он ДОЛЖЕН прочитать `.ai-team/artifacts/{feature}/design.md`
- **И** прочитать `.ai-team/artifacts/{feature}/tasks.md`
- **И** реализовать код в дереве исходников целевого проекта
- **И** написать unit-тесты для нового кода

#### Сценарий: Без отклонений от спецификации
- **КОГДА** Coder реализует
- **ТОГДА** он НЕ ДОЛЖЕН добавлять фичи или поведение, не описанные в дизайне и specs

### Требование: Coder использует AgentCLI runtime
Coder ДОЛЖЕН делегировать фактическую запись кода OpenCode CLI.

#### Сценарий: Реализация через OpenCode
- **КОГДА** Coder запускается
- **ТОГДА** система ДОЛЖНА подготовить промпт с контекстом из design.md и tasks.md
- **И** запустить `opencode --resume --message-file` в целевом проекте
- **И** OpenCode ДОЛЖЕН записать фактические файлы исходников
