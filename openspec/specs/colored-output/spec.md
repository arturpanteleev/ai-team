## Purpose

Спецификация определяет нормативное поведение capability `colored-output`.

## Requirements
### Requirement: ANSI-цветной вывод
Консольный вывод системы MUST использовать ANSI-цвета для ключевых элементов.

#### Scenario: Цветной вывод статусов
- **КОГДА** система выводит статус этапа в терминал
- **ТОГДА** `✓` MUST быть зелёным (`\033[32m`)
- **И** `✗` MUST быть красным (`\033[31m`)
- **И** название этапа MUST быть жёлтым (`\033[33m`)

### Requirement: Автоотключение цветов
Цвета MUST автоматически отключаться при перенаправлении вывода.

#### Scenario: Цвета не выводятся в файл
- **КОГДА** stdout перенаправлен в файл или pipe
- **ТОГДА** ANSI-escape последовательности MUST NOT выводиться
- **И** весь текст MUST выводиться как plain text
